0. Downloading certificate queue FPTestcert4_20220818.p12 from https://www.bankid.com/en/utvecklare/test.

2. Delleting unnecessary sertificate instance:
keytool -delete -alias mycertificate -keystore keystorefile.jks

3. Import sertificates from FPTestcert4_20220818.p12 to created Key Store:
keytool -v -importkeystore -srckeystore FPTestcert4_20220818.p12 -srcstoretype PKCS12 -destkeystore keystorefile.jks -deststoretype JKS

4. Imported certificate data check:
keytool -v -list -keystore keystorefile.jks
